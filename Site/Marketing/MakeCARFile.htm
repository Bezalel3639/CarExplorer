How to Make CAR File

Notice that MSB Wizard creates MSS file, but Car Explorer reads CAR files. It should not bother you since MSS file format is exactly the same as CAR file format if used for car descriptions. Simply rename resulting MSS file exptension to CAR extension manually after MSB Wizard makes the file.

  • Download MSB Wizard (component of MyScreenSaver Builder) for Windows 7/Vista or Windows XP, install, run MSB Wizard:



  • On the second Wizard's page point to the location of car pictures (JPG or GIF):



  • Click Next:



  • Double-click a car you want to add quick specications (you can skip it, and edit CAR file at any time):



  • Fill out the form for the car either manually:



  • Or, using XML file (populate from XML button) in the format (must be ANSI):


    • How to Edit CAR File

    With edit options you can (1) add car descriptions, (2) edit car description, (3) remove car descriptions.


    Notice that MSS file format is exactly the same as CAR file format if used for car descriptions. You need to manually rename CAR file exptension to MSS extension before using MSB Wizard.

  • Run MSB Wizard, click Next in the Welcome page, then point to the location of the CAR file you want to edit with Browse or simply drag-and-drop the file:



  • Make sure that "Edit" option is selected, and use "Edit image set data" in the edit options area, then click Next:



    • Now you can add, modify, or remove car descriptions:

    • Adding car description. Start with JPG or GIF car picture and drag-and-drop it on the list. Double click on the added item to run the dialog where you can add text specifications. Generally JPG or GIF should be less than 50-100KB to keep resulting CAR file about 1MB or less.
    • Editing car description. Double click an item you want to edit. Edit textual descriptions or replace picture.
    • Removing car description. Select an item and click Delete.
